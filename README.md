# Handgesture

Based on [**Andreas Schallwig**'s rock-paper-scissors game'](https://github.com/andypotato/rock-paper-scissors?ref=hackernoon.com).
The goal are:

* Isolate the important parts
* Make a reusable utility



## Usage

### Install

```bash
npm i
```

### Run

```bash
npm run dev
```
